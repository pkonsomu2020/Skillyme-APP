## 📖 **Overview**

Skillyme Africa is a comprehensive founder ecosystem and launchpad designed to bridge the gap between ambitious builders and world-class tech education. Our platform facilitates meaningful guidance through intensive programs, interactive LMS portals, and direct access to mentors and resources.

### 🎯 **Mission**
To empower the next generation of African founders and engineers by providing direct access to cutting-edge AI programs, a robust LMS portal, and networking opportunities that accelerate professional development.

### 🌟 **Vision**
To become the leading launchpad in Kenya and Africa, incubating the continent's most promising startups and engineers.

## 🏗️ **Architecture**

### **Frontend Applications**
- **Main Landing Page** (`www.skillyme.africa`): The gateway to our programs, mission, and community.
- **Programs Portal** (`program.skillyme.africa`): A dedicated portal for exploring, enrolling, and managing specific cohort programs and bootcamps.
- **LMS Portal** (`lmsportal.skillyme.africa`): An interactive Learning Management System designed to deliver course content, track progress, and facilitate peer-to-peer learning.

## 🚀 **Getting Started**

### **Prerequisites**
- Node.js 18.0.0 or higher
- npm, yarn, pnpm or bun package manager

### **Installation**

#### **1. Clone Repository**
```bash
git clone https://github.com/your-username/skillyme-africa-launchpad.git
cd skillyme-africa-launchpad
```

#### **2. Install Dependencies**
```bash
npm install
# or
bun install
```

#### **3. Start Development Server**
```bash
npm run dev
# or
bun dev
```
